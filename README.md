# Full-Stack Development Guide 

Installation Guides For Full-Stack Developers Using MAC OS 

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/o2team/mac.git
$ cd mac
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server --watch
```
